Homebrew的在mac安装

发表于2021-04-23|更新于2022-08-25
|字数总计:221|阅读时长:1分钟|阅读量:

1.Homebrew的安装（镜像一般选1（中科大））

/bin/zsh -c “$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)"

2.安装Nginx（在安装完Homebrew后重新打开终端）

brew install nginx

3.Nginx的一些常用命令

1.查看 nginx 版本

nginx -v

2.启动 nginx

sudo nginx

3.关闭nginx

sudo nginx -s stop

4.重新加载nginx

sudo nginx -s reload

5.可能遇到的问题
端口被占用

nginx: [emerg] bind() to 0.0.0.0:80 failed (48: Address already in use)

解决方法：修改 nginx.conf 文件里的端口号

权限不够

nginx: [alert] could not open error log file: open() “/usr/local/var/log/nginx/error.log” failed (13: Permission denied)

解决方法：在命令前加上 sudo，这时可能会要求输入密码，密码就是电脑的开机密码啦~

备注：端口号是在配置文件 nginx.conf 里面配置的，默认端口是 8080 ，配置文件的位置 /usr/local/etc/nginx
